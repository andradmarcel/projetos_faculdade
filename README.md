Super Trunfo em C
Este repositório contém a implementação de um jogo de Super Trunfo em linguagem C, desenvolvido como projeto de faculdade para praticar lógica de programação e manipulação de dados no terminal.
​

Sobre o jogo
Super Trunfo é um jogo de cartas em que cada carta possui vários atributos numéricos, e os jogadores comparam um atributo escolhido para decidir quem vence a rodada.
​
O objetivo é vencer o máximo de rodadas possível a partir da comparação desses atributos entre a carta do jogador e a carta do oponente.
​

Tecnologias
Linguagem C para implementação da lógica do jogo.
​

Execução via terminal/linha de comando.

Objetivo do projeto
Exercitar conceitos básicos de programação em C, como variáveis, estruturas, vetores, condicionais e laços de repetição.
​

Simular, de forma simples, a mecânica de um jogo de cartas do tipo Super Trunfo.
